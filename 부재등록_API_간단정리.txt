[부재등록 API/DTO 최소 요약]

1. 등록(POST)
  /attendance/absence
  {
    type: 출장|연수|연차|반차|외출|기타
    startDate: YYYY-MM-DD
    endDate: YYYY-MM-DD
    time: 09:00 (옵션)
    reason: string
  }

2. 내 목록조회(GET)
  /attendance/absence/my
  -> [{ id, type, startDate, endDate, time, reason }]

3. 상세조회(GET)
  /attendance/absence/{absenceId}

4. 수정(PUT)
  /attendance/absence/{absenceId}
  {
    type, startDate, endDate, time, reason
  }

5. 삭제(DELETE)
  /attendance/absence/{absenceId}

[DTO 예시]
class AbsenceDto {
  String type;
  String startDate;
  String endDate;
  String time; // 옵션
  String reason;
}
